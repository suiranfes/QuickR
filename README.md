# 翠巒祭 販売システム - Quick QR
sell.suiranfes.blue

## 技術
- メインライブラリ: React
- 言語: TypeScript
- データベース: Google Spread Sheet
- デザイン: Material UI
- フォント: IBM Plex Sans, IBM Plex Sans JP

### 動作方法
1. [*node.js* と *npm* をインストール](https://nodejs.jp/)

1. 依存ライブラリをインストール
    ```shell
    $ npm i
    ```

1. アプリケーションを開発者モードで動作
    ```shell
    $ npm start
    ```

1. ブラウザでアクセス  
ローカルホスト: http://localhost:3000
